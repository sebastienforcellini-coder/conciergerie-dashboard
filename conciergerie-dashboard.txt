Résumé complet — Projet conciergerie-dashboard
Contexte
Application de gestion pour une conciergerie gérant 40 riads/maisons au Maroc, avec 25+ propriétaires différents. Basée sur le projet Riad Dashboard existant. Rémunération par commission % sur chaque réservation.

React + Vite (vite v8)
Firebase Firestore — base de données
react-router-dom — navigation
recharts — graphiques
